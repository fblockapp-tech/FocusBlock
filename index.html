FOCUSBLOCK Bloquea las distracciones. De verdad.

FocusBlock modifica tu archivo hosts para bloquear YouTube, Instagram, TikTok y cualquier sitio que te robe el tiempo. Sin suscripción. Sin procesos en segundo plano. Con código de desbloqueo por email para que no puedas hacer trampa.

Descargar para macOS ⊞ Descargar para Windows

Gratis · Sin registro · Código abierto · Python 3 requerido

focusblock — hosts — bash
$focusblock --activate --duration 25m
✓ Código enviado a tu@email.com
✓ Bloqueando 28 dominios...
$cat /etc/hosts | grep youtube
127.0.0.1 youtube.com
127.0.0.1 www.youtube.com
127.0.0.1 m.youtube.com
$focusblock --deactivate

Simple. Efectivo. Sin trucos.

No instala extensiones, no crea procesos en segundo plano, no necesita internet para funcionar. Solo modifica un archivo del sistema.

// 01
Descarga y abre
Descarga el archivo, ejecuta el instalador y abre FocusBlock desde tu Escritorio. Solo necesitas Python 3 instalado.
// 02
Pon tu email
Introduce el email donde quieres recibir el código de desbloqueo. Puede ser cualquier cuenta: Gmail, Outlook, lo que uses.
// 03
Activa el foco
Elige la duración (o sin límite) y pulsa Activar. Los sitios quedan bloqueados instantáneamente en todos los navegadores.
// 04
El código te frena
Para desactivar necesitas el código que llegó a tu email. Si el móvil también está lejos... no hay escapatoria.

🔒
Bloqueo real a nivel de sistema
Modifica el archivo hosts del sistema operativo. Funciona en Chrome, Firefox, Safari, Edge y cualquier app que use DNS.
📧
Código de desbloqueo por email
Al activar, se envía un código a tu correo. Sin él, no puedes desactivar el bloqueo. El sistema te protege de ti mismo.
⏱
Temporizador Pomodoro
25 min, 45 min, 1h, 2h, 4h o sin límite. Al acabar el tiempo, el bloqueo se desactiva automáticamente.
✏️
Lista personalizable
Añade o quita cualquier dominio. La lista se guarda localmente en tu ordenador. Tus datos no salen de tu máquina.
🖥
macOS y Windows
Versión nativa para macOS (11+) y Windows 10/11. Misma experiencia, mismo diseño, en ambos sistemas.
🔍
Código abierto y auditable
El código fuente es Python puro, visible y auditable. No hay binarios opacos ni procesos ocultos.
